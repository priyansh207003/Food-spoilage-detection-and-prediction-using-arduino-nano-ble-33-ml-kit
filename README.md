# 🍎 Edge-Based Food Spoilage Prediction & Detection System

A TinyML-based food freshness detection system developed using the **Arduino Nano 33 BLE Sense** and **Edge Impulse**.

## 🚀 Overview

This project analyzes fruits and vegetables in real time and classifies them into:

* 🟢 Fresh
* 🟡 Semi-Rotten
* 🔴 Rotten

The model was trained on **10,000+ images** and performs on-device inference to estimate food freshness and remaining shelf life. Prediction results are displayed on an Android application via Bluetooth.

## 🛠️ Tech Stack

* Arduino Nano 33 BLE Sense
* Edge Impulse
* TinyML
* Machine Learning
* Computer Vision
* Android Studio
* Bluetooth
